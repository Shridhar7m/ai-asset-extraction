# AI Asset Information Extraction System

## 📌 Objective
This project is a simplified AI-powered system that extracts structured product information using web search and a language model approach.

---

## ⚙️ How It Works
1. Accepts input (model number, asset classification, manufacturer)
2. Performs web search using SerpAPI
3. Extracts relevant context
4. Uses an LLM (mocked in this version) to generate structured output
5. Validates and retries if needed
6. Returns final JSON output
